                         Elden Ring AIO Launcher

Thank you for downloading the Elden Ring AIO (All-In-One) Launcher!

This launcher provides a beautiful, native user interface that allows you to easily switch between Vanilla (Online), Offline mode, and Seamless Co-op without having to manually swap files or run separate launchers.

--------------------------------------------------------------------------------
 INSTALLATION INSTRUCTIONS
--------------------------------------------------------------------------------

1. Extract the `EldenRingAIOLauncher.exe` file from this ZIP.
2. Place the executable anywhere on your computer (e.g., your Desktop).
3. Run the launcher. 
4. The first time you run it, it will ask you to locate your game's `start_protected_game.exe` file.
   - Typically found in: C:\Program Files (x86)\Steam\steamapps\common\ELDEN RING\Game\
5. Select the file. The launcher will automatically back up your original executable and install itself in its place.
6. You're done! From now on, whenever you click "Play" in Steam, this custom launcher will appear.

--------------------------------------------------------------------------------
 FEATURES & MODES
--------------------------------------------------------------------------------

- VANILLA (ONLINE): Boots the game normally with Easy Anti-Cheat enabled for official servers.
- SEAMLESS CO-OP: Automatically detects and launches the Seamless Co-op mod with Steam tracking disabled for a clean injection (requires the Seamless Co-op mod files to be installed in your Game folder).
- OFFLINE MODE: Launches the game directly without Easy Anti-Cheat.
- CUSTOM APPS: You can add any custom .exe file (like ModEngine2 or other tools) directly to the launcher's list using the "+ Add" button.

--------------------------------------------------------------------------------
 UNINSTALLING
--------------------------------------------------------------------------------

To completely remove the launcher and restore your game to its original state:
1. Open the launcher (e.g., by clicking Play on Steam).
2. Click the red "Uninstall" button at the bottom left.
3. Confirm the prompt.
4. The launcher will safely restore your original executable and completely delete itself. No trace left behind!
